About This Course

CEUE203 introduces students to the Object Oriented Programming paradigm using Java. The course covers Java fundamentals, class mechanics, string handling, inheritance and interfaces, exception handling, multithreading, file I/O with NIO.2, and the Collections Framework with Generics.

Sessions combine lecture-based theory (2 hrs/week) with hands-on lab practicals (2 hrs/week), building a MiniBank project incrementally across 14 practicals that cover the full OOP lifecycle — from basic classes through concurrency, persistence, and testing.
