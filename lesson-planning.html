Lesson Planning

Unit-wise lecture session plan for CEUE203 – Object Oriented Programming

CEUE203 3rd Semester · 2026–27 8 Units · 30 Lecture Hours Faculty: Prof. Sagar Patel

Textbook key — Chapters Covered column:    T1 = Java: The Complete Reference, 13th Ed. — Herbert Schildt (McGraw Hill)  ·  T2 = OCP Java SE 17 Developer Programmer’s Guide — Mughal & Strelnikov (Oracle Press)

Unit 1 Fundamental of Programming in Java 2 hrs  ·  CO1  ·  Week 1  ·  6%

Lecture Week Topic Teaching Method CO Chapters Covered
L1 1 Java overview and architecture: history of Java, Java buzzwords (simple, object-oriented, robust, multithreaded, architecture-neutral, portable), JVM / JRE / JDK, bytecode and the ‘write once, run anywhere’ model Lecture + Discussion CO1 T1: Ch. 1–2
L2 1 Program structure and recent features: main method, compiling and running, switch expressions, records, enums and var Lecture + Demo CO1 T1: Ch. 2, 17

Unit 2 Class Fundamentals 4 hrs  ·  CO1  ·  Weeks 2–3  ·  14%

Lecture Week Topic Teaching Method CO Chapters Covered
L3 2 Classes and objects: defining classes, creating objects with new, instance vs static variables and methods, access modifiers (public, private, protected, default) Lecture + Demo CO1 T1: Ch. 6
L4 2 Constructors and keywords: default and parameterized constructors, constructor overloading, this and this() call, final with variables/methods/classes, instanceof Lecture + Examples CO1 T1: Ch. 6–7
L5 3 Nested and inner classes: static nested class, inner class, method-local inner class, anonymous inner class; Object class methods: toString(), equals(), hashCode(), clone(); overriding toString() and equals() Lecture + Code Demo CO1 T1: Ch. 7, 12
L6 3 Memory management, annotations and reflection: stack vs heap, object lifecycle and garbage collection; annotations and meta-annotations; custom annotations; reflection Lecture + Examples CO1 T1: Ch. 12–13

Unit 3 String Handling 2 hrs  ·  CO1  ·  Week 4  ·  6%

Lecture Week Topic Teaching Method CO Chapters Covered
L7 4 String handling: String class methods, StringBuffer and StringBuilder, string manipulation and immutability Lecture + Examples CO1 T1: Ch. 18
L8 4 Regular expressions and tokenization: Pattern and Matcher, common regex patterns, string tokenization with split() Lecture + Code Demo CO1 T1: Ch. 16, 18

Unit 4 Inheritance, Interfaces & Packages 5 hrs  ·  CO2  ·  Weeks 5–7  ·  17%

Lecture Week Topic Teaching Method CO Chapters Covered
L9 5 Inheritance and polymorphism: extends, super keyword, dynamic method dispatch and runtime polymorphism, final in inheritance Lecture + Examples CO2 T1: Ch. 8
L10 5 Abstraction: abstract classes and methods, upcasting and downcasting objects in inheritance Lecture + Code Demo CO2 T1: Ch. 8
L11 6 Interfaces: declaring and implementing interfaces, Interface Segregation Principle, default, static and private interface methods Lecture + Examples CO2 T1: Ch. 9
L12 6 Advanced interfaces: multiple inheritance via interfaces, functional interfaces and lambdas, marker interfaces Lecture + Code Demo CO2 T1: Ch. 9, 15
L13 7 Packages and modules: creating and importing packages, access across packages, static imports, JAR files, Java Module System (modules, descriptors, services, service loaders) Lecture + Demo CO2 T1: Ch. 9, 16

Unit 5 Exception Handling 2 hrs  ·  CO3  ·  Weeks 7–8  ·  6%

Lecture Week Topic Teaching Method CO Chapters Covered
L14 7 Exception handling basics: checked vs unchecked exceptions, try-catch-finally, exception class hierarchy Lecture + Examples CO3 T1: Ch. 10
L15 8 Advanced exception handling: throw and throws, custom exceptions, try-with-resources, exception propagation Lecture + Code Demo CO3 T1: Ch. 10, 13

Unit 6 Multithreaded Programming 5 hrs  ·  CO4  ·  Weeks 8–10  ·  17%

Lecture Week Topic Teaching Method CO Chapters Covered
L16 8 Introduction to multithreading: processes vs threads, life cycle of a thread, ways to create threads Lecture + Discussion CO4 T1: Ch. 11
L17 9 Creating threads: Thread class and Runnable interface, thread groups, start() vs run() Lecture + Code Demo CO4 T1: Ch. 11
L18 9 Thread synchronization: synchronized methods and blocks, race conditions, inter-thread communication (wait/notify) Lecture + Examples CO4 T1: Ch. 11
L19 10 Thread coordination: thread priority and scheduling, deadlock and thread-safety Lecture + Demo CO4 T1: Ch. 11
L20 10 Executors and parallelism: Executor and ExecutorService for thread-pool management, Fork/Join framework Lecture + Code Demo CO4 T1: Ch. 11

Unit 7 File NIO 4 hrs  ·  CO5  ·  Weeks 11–12  ·  14%

Lecture Week Topic Teaching Method CO Chapters Covered
L21 11 File handling with NIO.2: Path interface, Files class, file attributes Lecture + Demo CO5 T1: Ch. 22–23
L22 11 Streams and serialization: byte and character streams, object serialization and deserialization Lecture + Code Demo CO5 T1: Ch. 22
L23 12 NIO fundamentals: Buffer, Channel and Selector Lecture + Examples CO5 T1: Ch. 23
L24 12 Directory operations: directory streams and file walking Lecture + Demo CO5 T1: Ch. 23

Unit 8 Collection Framework and Generics 6 hrs  ·  CO6  ·  Weeks 13–15  ·  20%

Lecture Week Topic Teaching Method CO Chapters Covered
L25 13 Collections and List: Collection Framework overview, Collection interface, List implementations (ArrayList, LinkedList, Vector, Stack) Lecture + Examples CO6 T1: Ch. 20
L26 13 Set: HashSet, LinkedHashSet, TreeSet and their use cases Lecture + Code Demo CO6 T1: Ch. 20
L27 14 Map: HashMap, LinkedHashMap, TreeMap, Hashtable Lecture + Examples CO6 T1: Ch. 20
L28 14 Working with collections: iterating through collections, sorting and searching, Comparable and Comparator, thread-safe collections Lecture + Demo CO6 T1: Ch. 20
L29 15 Generics: generic classes and methods, bounded type parameters Lecture + Examples CO6 T1: Ch. 14
L30 15 Advanced generics and revision: wildcards, type erasure, course revision and integrated application walkthrough Problem Solving / Review CO6 T1: Ch. 14–15
