About This Course

CEUE203 introduces students to the Object Oriented Programming paradigm using C++. The course covers foundational concepts of classes, objects, encapsulation, inheritance, polymorphism, exception handling, file I/O, and the Standard Template Library (STL).

Sessions combine lecture-based theory with hands-on lab practicals, building a cohesive understanding of OOP design principles and their application to real-world software problems.
