Course Outcomes (CO)

At the end of this course, students will be able to:

CO Statement Bloom's Level
CO1 Understand and explain the fundamental concepts of Object Oriented Programming including encapsulation, abstraction, and the advantages of OOP over procedural programming. Understand (L2)
CO2 Design and implement classes and objects with constructors, destructors, member functions, access specifiers, and static members in C++. Apply (L3)
CO3 Apply inheritance mechanisms including single, multiple, multilevel, and hierarchical inheritance to model real-world relationships. Apply (L3)
CO4 Implement polymorphism through function overloading, operator overloading, and virtual functions to achieve runtime behavior. Apply (L3)
CO5 Implement exception handling mechanisms and perform file I/O operations using streams for robust program development. Apply (L3)
CO6 Utilize function templates, class templates, and Standard Template Library (STL) containers and algorithms for generic and efficient programming. Analyse (L4)
